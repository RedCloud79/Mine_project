# 주요기능 
카메라 만을 활용하였을 경우의 문제점

모터의 딜레이를 주어서 속도를 낮춰서 카메라 포커스 안에 들어오게 개선을 했지만, 라인을 인식하는 부분과 사람을 인식하는 부분이 동시에 동작이 되지 않음.
사람을 인식받았을경우 우회해서 다시 라인으로 들어오는 코드를 사용할경우 사람의 객체인식의 데이터가 많아서 반복적으로 우회하는 코드가 실행되어 의도한 경우와 다르게 동작된다.
